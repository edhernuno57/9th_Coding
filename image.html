Understanding the <img> Tag

The `` tag in HTML is used to embed images in a webpage. It is a self-closing tag, meaning it does not require a closing tag. The `` tag has several important attributes that you should know about:

The `src` Attribute

The `src` (source) attribute is used to specify the path to the image you want to display. This can be a relative path (if the image is stored in the same directory as your HTML file) or an absolute path (a full URL to an image hosted online).

Example:

<img src="image.jpg" alt="A beautiful scenery">

The `alt` Attribute

The `alt` (alternative text) attribute is used to provide a text description of the image. This text is displayed if the image cannot be loaded and is also used by screen readers for accessibility purposes.

Example:

<img src="image.webp" alt="A beautiful scenery">

The `width` and `height` Attributes

The `width` and `height` attributes are used to specify the dimensions of the image in pixels. You can use these attributes to resize the image to fit your webpage layout.

Example:

<img src="image.jpg" alt="A beautiful scenery" width="200" height="100">

Combining Attributes

You can combine these attributes to fully customize how the image is displayed on your webpage.

Example:

<img src="image.jpg" alt="A beautiful scenery" width="300" height="200">
