Understanding <ul> and <li> Tags

The <ul> tag in HTML is used to create an unordered list, where each item is marked with a bullet point. The <li> tag is used to define each list item within the unordered list.

Basic Example

Hereâ€™s an example of an unordered list with three items:

Input HTML Code:

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

Output:

  • First item
  • Second item
  • Third item

In the example above, the <ul> tag creates the unordered list, and each item is defined by the <li> tag. The items are automatically marked with bullet points.

Customizing the List

You can customize the style of the bullet points in an unordered list by using the type attribute in the <ul> tag. Here are some examples:

Input HTML Code:

<ul type="disc">
  <li>Bulleted with discs (default)</li>
</ul>

<ul type="circle">
  <li>Bulleted with circles</li>
</ul>

<ul type="square">
  <li>Bulleted with squares</li>
</ul>

Output:

  • Bulleted with discs (default)
  • Bulleted with circles
  • Bulleted with squares

By using the list-style-type CSS property, you can change the bullet style to discs, circles, or squares.
