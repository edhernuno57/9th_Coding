Understanding <ol> and <li> Tags

The <ol> tag in HTML is used to create an ordered list, where each item is automatically numbered. The <li> tag is used to define each list item within the ordered list.

Basic Example

Hereâ€™s an example of an ordered list with three items:

Input HTML Code:

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Output:

  1. First item
  2. Second item
  3. Third item

In the example above, the <ol> tag creates the ordered list, and each item is defined by the <li> tag. The items are automatically numbered 1, 2, 3, and so on.

Customizing the List

You can customize the type of numbering used in an ordered list by using the type attribute in the <ol> tag. Here are some examples:

Input HTML Code:

<ol type="1">
  <li>Numbered with digits (1, 2, 3...)</li>
</ol>

<ol type="A">
  <li>Numbered with uppercase letters (A, B, C...)</li>
</ol>

<ol type="a">
  <li>Numbered with lowercase letters (a, b, c...)</li>
</ol>

<ol type="I">
  <li>Numbered with uppercase Roman numerals (I, II, III...)</li>
</ol>

<ol type="i">
  <li>Numbered with lowercase Roman numerals (i, ii, iii...)</li>
</ol>

Output:

  1. Numbered with digits (1, 2, 3...)
  1. Numbered with uppercase letters (A, B, C...)
  1. Numbered with lowercase letters (a, b, c...)
  1. Numbered with uppercase Roman numerals (I, II, III...)
  1. Numbered with lowercase Roman numerals (i, ii, iii...)

By using the type attribute, you can change the numbering style to digits, letters, or Roman numerals, depending on your needs.
