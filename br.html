Understanding the <br> Tag

The <br> tag in HTML is used to insert a line break. Unlike the <p> tag, which creates a new paragraph with space above and below, the <br> tag simply moves the text that follows it to a new line, without any additional spacing.

The <br> tag is an empty element, meaning it doesn't have a closing tag. It is self-closing and is written as <br> or <br />.

Basic Example

This is the first line.
This is the second line.

In the example above, the text "This is the first line." is followed by a <br> tag, which moves the text "This is the second line." to the next line.

Using Multiple Line Breaks

This is the first line.

This is the second line after two breaks.

You can use multiple <br> tags in a row to create additional space between lines, as shown in the example above.

When to Use <br> vs <p>

Use the <br> tag when you need to create a line break within the same paragraph or block of text, such as in addresses, poems, or where you want to break lines without creating a new paragraph.

For example:

123 Main Street
Springfield, USA

On the other hand, use the <p> tag when you want to create a new paragraph with spacing above and below.

Conclusion

The <br> tag is a simple but useful tool for controlling the layout of text on a webpage, allowing you to insert line breaks wherever necessary without starting a new paragraph.
