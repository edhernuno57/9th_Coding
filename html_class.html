Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages. HTML elements are the building blocks of all websites.

Common HTML Tags

  • <h1> to <h6> - Headers: Used to create headings, with <h1> being the largest and <h6> the smallest.
  • <p> - Paragraph: Used to define a paragraph of text.
  • <br> - Break Line: Inserts a line break in the text.
  • <ol> - Ordered List: Creates a numbered list of items.
  • <ul> - Unordered List: Creates a bullet-point list of items.
  • <img> - Image: Embeds an image in the webpage. Requires the 'src' attribute to specify the image source.
  • <a> - Anchor: Creates a hyperlink to another webpage. The 'href' attribute specifies the link's destination.
  • <button> - Button: Creates a clickable button.
  • <div> - Division: Used to group elements together in a block for styling purposes.
