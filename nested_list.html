Understanding Nested Lists in HTML

In HTML, you can create nested lists by placing a list (either ordered or unordered) inside another list. This allows you to create complex hierarchical structures.

Basic Example of a Nested Unordered List

Here’s an example of a nested unordered list:

Input HTML Code:

<ul>
  <li>Item 1</li>
  <li>Item 2
    <ul>
      <li>Subitem 2.1</li>
      <li>Subitem 2.2</li>
    </ul>
  </li>
  <li>Item 3</li>
</ul>

Output:

  • Item 1
  • Item 2
    • Subitem 2.1
    • Subitem 2.2
  • Item 3

In the example above, the nested list appears as a list within a list, with subitems indented and displayed under "Item 2".

Nested Ordered List Inside an Unordered List

You can also mix ordered and unordered lists in a nested structure. Here’s an example:

Input HTML Code:

<ul>
  <li>Item 1</li>
  <li>Item 2
    <ol>
      <li>Subitem 2.1</li>
      <li>Subitem 2.2</li>
    </ol>
  </li>
  <li>Item 3</li>
</ul>

Output:

  • Item 1
  • Item 2
    1. Subitem 2.1
    2. Subitem 2.2
  • Item 3

In this example, "Item 2" contains a nested ordered list, where subitems are numbered.

Nested Unordered List Inside an Ordered List

You can also have an unordered list inside an ordered list. Here’s how:

Input HTML Code:

<ol>
  <li>Step 1</li>
  <li>Step 2
    <ul>
      <li>Option 2.1</li>
      <li>Option 2.2</li>
    </ul>
  </li>
  <li>Step 3</li>
</ol>

Output:

  1. Step 1
  2. Step 2
    • Option 2.1
    • Option 2.2
  3. Step 3

Here, "Step 2" contains an unordered list, providing options within the step.
